This is an express-starter-template.

node packages used:
	bcrypt
	bluebird
	body-parser
	connect-flash
	ejs
	express
	express-session
	mongoose
	passport
	passport-local

Open the terminal

cd Documents/express-starter-template
npm install

Run MongoDB

mongod

Open new terminal tab

Start the server

npm server.js

Run the project
1. Open a browser
2. Go to localhost:3000

The project should now be running in the browser.
